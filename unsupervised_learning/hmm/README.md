# Hidden Markov Models

This directory contains implementations of Hidden Markov Model algorithms.

## Tasks

### 0. Markov Chain
Determines the probability of a Markov chain being in a particular state after t iterations.

**File:** `0-markov_chain.py`

### 1. Regular Chains
Determines the steady state probabilities of a regular Markov chain.

**File:** `1-regular.py`

### 2. Absorbing Chains
Determines if a Markov chain is absorbing.

**File:** `2-absorbing.py`

### 3. The Forward Algorithm
Performs the forward algorithm for a hidden Markov model.

**File:** `3-forward.py`

### 4. The Viterbi Algorithm
Calculates the most likely sequence of hidden states using the Viterbi algorithm.

**File:** `4-viterbi.py`

### 5. The Backward Algorithm
Performs the backward algorithm for a hidden Markov model.

**File:** `5-backward.py`

### 6. The Baum-Welch Algorithm
Performs the Baum-Welch (EM) algorithm to learn HMM parameters from observations.

**File:** `6-baum_welch.py`
